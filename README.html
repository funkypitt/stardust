We Are Stardust

The "We Are Stardust" Commodore 64 release, packaged as a stand-alone Android app.

Overview

This is a single-purpose Android app that ships an embedded Commodore 64 emulator (Frodo by Christian Bauer) pre-configured to autoload We Are Stardust (M).d64 from app/src/main/assets/. The native emulator core lives under app/src/main/cpp/emu/ and is built via NDK / CMake; the Java side wraps it with a touch UI, an action strip, and a fullscreen landscape activity.

Features

  • C64 emulator (Frodo) embedded in-app, no external ROM picking required
  • Autoloads the bundled We Are Stardust (M).d64 on launch
  • Fullscreen landscape, on-screen action strip and touch input
  • Native cores for armeabi-v7a, arm64-v8a, x86, x86_64
  • Min SDK 21, target SDK 34

Screenshots

No screenshots/ folder is shipped with the source.

Build & install

Standard Android Gradle project with an NDK / CMake C++ component.

./gradlew assembleRelease

The C++ flags are set in app/build.gradle: -frtti -fexceptions -Wno-c++11-narrowing -Wno-writable-strings -Wno-dangling-else.
